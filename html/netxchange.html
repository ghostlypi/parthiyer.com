NetXchange is a sunsetted project which aimed to enable end-to-end encrypted asynchronous data transfers via 3rd party servers. The goal is to act like a VPN similar to tailscale, enabling secure direct communication between devices this project was proof of concepted initially with TLS and later with the NIST approved CRYSTALS Kyber algorithms for Quantum Resistant encrypted messaging between clients.
